SOMBRAS

BOX-SHADOW

Cria uma sombra. No Exemplo a seguir utilizei: box-shadow: 10px 10px 10px 1px darkgoldenrod; Onde a ordem das propriedades é horizontal, vertical, borrão, preenchimento, cor da sombra.

INSET

Cria uma sombra interna. No Exemplo a seguir utilizei: box-shadow: inset 10px 10px 10px 1px darkgoldenrod; Onde a ordem das propriedades é a mesma.

SOMBRA DUPLA

Para criar uma sombra dupla, eu utilizei: box-shadow: 10px 10px 10px 1px darkgoldenrod, 10px 10px 10px 10px darkcyan; Isto serve para ter várias bordas, tanto interna, quanto externa, é só separar cada sombra por vírgula (,).

DROP-SHADOW

Para criar uma sombra ao redor da imagem png, eu utilizei: filter: drop-shadow (10px 10px 5px darkblue);

TEXT-SHADOW

Para criar uma sombra ao redor do texto, eu utilizei: text-shadow: 10px 10px 5px black;

Caso eu não colocasse cor para sombra, ela seria a mesma da letra.
OPACITY

Para criar uma transparência no elemento, se utiliza: opacity: e a porcentagem de opacidade do elemento, indo de 1 para 100% 0 para 0% 0.1 até 0.9 para porcentagens decimais (10% a 90%) 0.01 até 0.09 para porcentagens centesimais (1% a 9%). 50% Comparação 100%
