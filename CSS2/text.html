TEXTOS

TEXT-TRANSFORM
  • text-tranform: none;
      não muda nada.
  • text-tranform: capitalize;
      altera todas as primeiras letras para maiúsculo.
  • text-tranform: uppercase;
      altera todas as letras para maiúsculo.
    • text-transform: inherit
        altera as letras de acordo com a propriedade pai.
  • text-tranform: lowercase;
      ALTERA TODAS AS LETRAS PARA MINÚSCULO.
    • TEXT-ALIGN
  • text-align: left;
      o texto fica alinhado a esquerda do quadrado.
  • text-align: right;
      o texto fica alinhado a direita do quadrado.
  • text-align: center;
      o texto fica centralizado no quadrado.
  • text-align: justify;
      o texto fica distribuido de forma a ocupar todo o quadrado.
    • TEXT-DECORATION

    TODAS AS PROPRIEDADES ABAIXO SEGUEM O COMANDO TEXT-DECORATION-LINE

    • text-decoration: none;
        deixa o texto sem nenhuma linha.
    • text-decoration: underline;
        sublinha o texto.
    • text-decoration: line-through;
        risca o texto.
    • text-decoration: overline;
        coloca uma linha acima do texto.
      • TEXT-DECORATION-STYLE
    • text-decoration-style: solid;
        traça uma linha sólida.
    • text-decoration-style: double;
        traça uma linha dupla.
    • text-decoration-style: dotted;
        traça uma linha pontilhada.
    • text-decoration-style: dashed;
        traça uma linha tracejada.
    • text-decoration-style: wavy;
        traça uma linha ondulada.
      • TEXT-DECORATION-COLOR colore apenas a linha.
      TEXT-DECORATION-THICKNESS muda a espessura da linha. Aqui eu coloquei 10px.

        word-spacing define um espaçamento entre as palavras dos textos.
      SPACING
        letter-spacing define um espaçamento entre as letras dos textos.
        word-spacing define um espaçamento entre as palavras dos textos.
        WHITE-SPACE

        white-space: normal

        os espaços em branco são resumidos em apenas 1.

        white-space: nowrap

        os espaços em branco são resumidos em apenas 1 ignorando, inclusive a definição da caixa.

        white-space: pre

        todos os espaços em branco são mantidos

        white-space: pre-line

        as quebras de linha são mantidas, porém, os espaços em branco não.

        white-space: pre-wrap

        as quebras de linha são mantidas e os espaços em branco também.

        word-wrap:

        normal Na_quebra_de_linha_a_palavra ultrapassa_o_limite da_caixa.

        break-worda_o_limite da_caixa.

        Na_quebra_de_linha_a_palavra não_ultrapass

        word-break:

        normal Na_quebra_de_linha_a_palavra ultrapassa_o_limite da_caixa.

        break-all Na_quebra_de_linha_a_palavra não_ultrapassa_o_limite da_caixa, mas ignora a quebra de linha do texto.

        keep-all Na_quebra_de_linha_a_palavra ultrapassa_o_limite da_caixa, já que é uma regra da escrita oriental. それは東洋の文章のルールだから.

        writing-mode:

        horizontal-tb

        Na escrita o padrão ocidental é ler da esquerda pra direita, ou seja, o comando acima é o padrão do programa.

        vertical-lr

        Nesse comando, o texto passa a ser lido na vertical, mas mantido da esquerda pra direita.

        vertical-rl

        Nesse comando, o texto passa a ser lido na vertical, porém, da direita pra esquerda.

        overflow: hidden

        esconde o texto que ultrapassa o limite da caixa.

        CLIP

        ELLIPSIS
